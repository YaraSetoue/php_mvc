Esse é um projeto para estudo, onde eu fiz a estrutura MVC na mão.

# Requisitos

- Docker

> Importante não ter outro container rodando na porta 8080, pois é a porta que eu uso nesse projeto. Pode so trocar também.

# Passos
1. Ir para pasta docker
```bash
cd docker/
```
2. Rodar o comando abaixo:
```bash
docker compose up -d --build
```
3. Colocar no navegador: localhost:8080
